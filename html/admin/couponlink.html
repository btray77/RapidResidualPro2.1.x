Use the examples below to format your coupon code links
 
Direct root sales page: {{http_path}}/{{fileName}}/{{coupon}}
Direct individual product sales page: {{http_path}}/{{fileName}}/{{product_name}}/{{coupon}}
Root sales page with affiliate link: {{http_path}}/{{fileName}}/affiliate_username/{{coupon}}
Individual product sales page with affiliate link: {{http_path}}/{{fileName}}/affiliate_username/{{product_name}}/{{coupon}}
*** affiliate_username will need to be replaced by affiliate user names to work properly.
